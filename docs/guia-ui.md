# Guía de UI - EfectivoYa

## Paleta de Colores

```
#1f1f1f - Negro principal (fondos)
#e83733 - Rojo acento (CTA, urgencia)
#fefefe - Blanco (texto principal)
#acacae - Gris (texto secundario, deshabilitados)
#dc993c - Dorado (premium, éxito)
```

## Uso de Colores

| Elemento | Color |
|----------|-------|
| Fondo principal | `#1f1f1f` |
| Botón primario | `#e83733` |
| Botón secundario | `#dc993c` |
| Texto sobre oscuro | `#fefefe` |
| Texto deshabilitado | `#acacae` |

## Estados de Operaciones

| Estado | Color |
|--------|-------|
| Pendiente | `#dc993c` (dorado) |
| Aprobado | `#10B981` (verde) |
| Rechazado | `#e83733` (rojo) |

## Tipografía

```
Font: Inter (Web) / System Font (Mobile)

H1: 32px, bold
H2: 24px, bold
H3: 20px, semibold
Body: 16px, regular
Caption: 14px, regular
Small: 12px, regular
```

## Espaciado

```
Padding: 8px, 16px, 24px, 32px
Gap: 8px, 12px, 16px, 24px
Border Radius: 8px (cards), 4px (inputs)
```
